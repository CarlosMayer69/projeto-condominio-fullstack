Projeto condomínio

O sistema de condomínio realiza o cadastramento dos condôminos, das unidades do condomínio, das garagens e dos utensílios da cozinha.

O sistema realiza o sorteio das garagens de forma aleatória. 
Existem garagens que não são sorteadas. 
Por exemplo, as garagens do síndico e do subsíndico não são sorteadas, são garagens fixas. 
Existem também, alguns condôminos que só possuem um automóvel, portanto, eles não devem ir para sorteio, 
deve ser atribuída para esses uma garagens já pré-determinada. 

O sistema gerência uma espécie de chat ou grupo, onde os condôminos possam se comunicarem através do mesmo, 
para tratar de assuntos de interesse do condomínio ou dos condôminos. 

O sistema gera alguns documentos tais como: reserva do salão de festas, relação de utensílio disponíveis na cozinha do salão de festas, etc.
